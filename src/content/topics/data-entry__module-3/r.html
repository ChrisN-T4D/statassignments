Entering and Importing Data in R

Importing Data

Use read.csv() or read.xlsx() to import data files.

# Import CSV
data <- read.csv("mydata.csv")

# Import Excel
library(readxl)
data <- read_excel("mydata.xlsx")

Manual Data Entry

Create data frames directly in R:

data <- data.frame(
  age = c(25, 30, 35),
  score = c(85, 90, 88)
)
