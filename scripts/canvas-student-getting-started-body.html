What is Methods Market?

Methods Market is where you do Concept Review quizzes and Software Practice for this course. Canvas still holds your syllabus, due dates, grades, and module schedule — when a module lists a Methods Market assignment, open the link, complete the activity, then mark the Canvas item done.

Open Statistics course home (Methods Market)

One-time setup (do this first)

You only need to do these steps once at the start of the semester. After that, use the module links in Canvas each week.

Step 1 — Create your account

  1. Go to methods-market.clneu.com/auth.
  2. Click Create Account (or sign in if you already have one).
  3. Use an email you check regularly and a password you will remember (at least 6 characters).

Step 2 — Link your student key

Your instructor will give you a student key (example: 2026FA-X7K9M2). This links your account to the class roster so your practice is tracked without putting your name on analytics exports.

  1. While signed in, go to methods-market.clneu.com/claim.
  2. Select semester: Fall 2026 (or the active semester your instructor names).
  3. Enter your student key exactly as provided.
  4. Click Link My Profile.

Don't have a key yet? Ask your instructor — keys are usually sent by Canvas message or email in the first week.

Step 3 — Bookmark the course

Save this link for quick access:

  • Statistics — Methods Market home
  • Assignment help & tips

Step 4 — Install jamovi (before Module 3)

Software Practice uses jamovi (free). Download from jamovi.org and install before you start Module 3. R, SPSS, Excel, and Stata options are also available in Methods Market if you already use them.

Textbook

We use Learning Statistics with jamovi — read online free. Chapter links for each module appear on the Methods Market course home when you select a module.

Each week — how assignments work

  1. Open the module for the current week in Canvas.
  2. Click the Methods Market link(s), for example:
    • Module N: Concept Review (Methods Market) — multiple-choice practice on ideas from lecture and readings.
    • Module N: Software Practice (Methods Market) — hands-on steps in jamovi, R, SPSS, Excel, or Stata (modules 3–8).
  3. Complete the activity in Methods Market (sign in if prompted).
  4. Return to Canvas and mark the assignment Done or submit per your instructor's instructions.

Note: Canvas does not automatically know when you finish in Methods Market. You are responsible for completing both places.

Concept Review vs Software Practice

Activity What you do When
Concept Review Answer quiz questions; read feedback after each item. Modules 1–8
Software Practice Follow guided steps in statistical software (choose jamovi, R, etc.). Modules 3–8 only

Troubleshooting

I can open the page but my progress isn't saved
Sign in at /auth and complete /claim with your student key.
"Invalid student key" or "Already claimed"
Check for typos and the correct semester. If the key was already used, sign in with the account that claimed it, or contact your instructor for a new key.
Forgot password
Use Forgot password on the sign-in page at /auth.
Link opens the wrong module
Use the link from this week's Canvas module, not an old announcement.
Due dates and points
Always follow Canvas for deadlines and grading. Methods Market is the practice site only.
